# CuteWorkTimer
Сделала за часик через нейронку таймер для работы, надеюсь еще буду апдейтить его. Ему есть куда улучшаться)

Чтобы запустить просто кникните дважды по timer.py или пкм запустить с помощью питона. В будущем сделаю его удобнее и выложу exe`шник.

В последнем обновлении 16.04.26 Добавлены новые фичи и улучшен таймер.
#Новое и обновленное:
1. Таймер теперь можно полностью скрыть до 1 строчки и при необходимости раскрыть
2. Все системные настройки с раскрытой информации вынесены в настройки (шестеренка).
3. В настройках теперь можно указать свой график. Начало рабочего дня, конец. Отдельно можно указать для пятницы, со скольки обед и сколько он длится, а также входит ли обед в рабочее время.
4. В корректировке времени теперь не просто поле для ввода, а удобный интерфейс добавления часов и минут.
5. В статистике добавлен новый раздел "График" , там можно выбрать месяц в котором у вас есть записи о работе и посмотреть по дням сколько часов было отработано в виде красивого графика.
6. Добавлено всплывающее окошко об обеде или окончании рабочего дня с информацией об отработанном дне. (Здесь можно расписать куда больше, но мне лень.) Там еще фича что ты отмечаешь начал ли ты обед, скок он идет. А если эт оконец дня, то он тебя поздравляет и предлагает закончить, либо продолжить.

Подводя итог, теперь таймер более минималистичен, практичен и открыт к кастому рабочего времени под вас!
